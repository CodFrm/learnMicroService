## 构建
我全程使用docker进行部署和学习,首先需要构建两个微服务的镜像,分别在```auth/Dockerfile```,```post/Dockerfile```

然后运行```docker-compose up```,这里需要注意的是可能需要运行多次,请参考[微服务架构本地尝试(2)-api网关](doc/微服务架构本地尝试(2)-api网关.md)中的**安装**
